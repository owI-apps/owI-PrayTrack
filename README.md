# owI-PrayTrack 📱

## Fitur
- 📊 Dashboard tracking poin ibadah harian
- 🕌 Pencatatan sholat wajib & sunnah
- 📖 Tracking bacaan Al-Quran
- 💰 Kalkulator zakat mal & fitrah
- 🕐 Manajemen utang sholat & puasa

## Tech Stack
- Vanilla JavaScript (ES6 Modules)
- Tailwind CSS
- Lucide Icons
- LocalStorage API

## Installation
- Clone repository
- Buka `index.html` di browser
- Atau deploy ke GitHub Pages

## Dark Mode
- Toggle tema di sidebar
- Auto-save preferensi
- Smooth transition
